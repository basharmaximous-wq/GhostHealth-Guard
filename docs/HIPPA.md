What is HIPAA?
HIPAA stands for Health Insurance Portability and Accountability Act. It's a law from 1996 that protects private health details, called PHI,
from being shared without permission.

Why Fines Like $50,000?
Breaking HIPAA rules, like accidentally showing patient names or info, can cost money. Small mistakes might mean $100 to $50,000 per slip-up,
depending on if it was on purpose.

GitHub and HIPAA Problems
When coders use GitHub to share or update health apps, they might forget secret info (like passwords or patient names)
in the code. This is like leaving your diary open online—anyone could see it and that's a HIPAA no-no. Tools check code changes (PRs)
to catch these before they go public
