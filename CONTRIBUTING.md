# Contributing to GhostHealth Guard

We're excited that you're interested in contributing to GhostHealth Guard!

## How to Contribute
1. **Report Bugs:** Open an issue on GitHub.
2. **Suggest Features:** Open an issue with the [Feature] label.
3. **Submit PRs:**
   - Fork the repository.
   - Create a feature branch.
   - Run `cargo fmt` and `cargo clippy`.
   - Ensure all tests pass with `cargo test`.
   - Submit your pull request!

## Code of Conduct
Please be respectful and helpful in all interactions.
